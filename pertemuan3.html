Layout Website dengan CSS Flexbox & Grid

Apa itu Flexbox?

Flexbox adalah metode layout CSS untuk menyusun elemen dalam satu arah (baris atau kolom) dengan mudah, bahkan jika ukurannya tidak diketahui.

Apa itu CSS Grid?

CSS Grid adalah metode layout untuk menyusun elemen dalam dua arah (baris dan kolom sekaligus), cocok untuk membuat tata letak halaman yang kompleks.

Contoh Flexbox:

.container-flex {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.item {
    flex: 1;
    margin: 0 10px;
    padding: 15px;
    background-color: #0066cc;
    color: white;
}

Contoh CSS Grid:

.container-grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 15px;
}

.grid-item {
    padding: 15px;
    background-color: #0099ff;
    color: white;
}

Pertemuan 3: Layout Website dengan CSS Flexbox & Grid

Topik yang Dibahas

CSS Flexbox

  • Apa itu Flexbox?
  • Properti untuk Container (display, flex-direction, justify-content)
  • Properti untuk Item (flex-grow, flex-shrink, align-self)
  • Penggunaan Flexbox untuk Navigasi & Kartu

CSS Grid

  • Apa itu CSS Grid?
  • Properti untuk Container (grid-template-columns, grid-gap)
  • Properti untuk Item (grid-column, grid-row)
  • Penggunaan Grid untuk Tata Letak Utama Website

Contoh Flexbox & Grid

Contoh Flexbox untuk Navigasi:

/* Style Navigasi dengan Flexbox */
nav {
    background-color: #f39c12;
    padding: 15px;
}

.nav-list {
    display: flex;
    list-style: none;
    justify-content: space-around; /* Rata kanan-kiri */
    align-items: center; /* Rata vertikal */
    margin: 0;
    padding: 0;
}

.nav-list li a {
    color: white;
    text-decoration: none;
    font-weight: bold;
}

Contoh Grid untuk Layout Kartu:

/* Style Layout Kartu dengan Grid */
.card-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 20px; /* Jarak antar kartu */
    padding: 20px;
}

.card {
    background-color: white;
    border-radius: 8px;
    padding: 20px;
    box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

Kapan Menggunakan Flexbox atau Grid?

Flexbox

➡️ Cocok untuk layout satu dimensi (baris atau kolom saja)

➡️ Ideal untuk navigasi, tombol, atau menyejajarkan konten dalam kotak

CSS Grid

➡️ Cocok untuk layout dua dimensi (baris dan kolom sekaligus)

➡️ Ideal untuk tata letak utama website, galeri gambar, atau daftar kartu

Tugas Praktikum

Buatlah layout website sederhana dengan menggunakan Flexbox & Grid:

  1. Buat navigasi atas dengan Flexbox (rata kanan atau tengah)
  2. Buat bagian utama dengan Grid (2 kolom: konten utama dan sidebar)
  3. Buat daftar kartu materi dengan Grid yang responsif
  4. Buat footer dengan Flexbox untuk menyejajarkan teks dan ikon
  5. Simpan style di file style-layout.css di folder css/
