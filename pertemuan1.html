Pengenalan HTML dan Struktur Dasar Website

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan konten pada website.

Struktur Dasar HTML:

  • <!DOCTYPE html>: Mendefinisikan dokumen sebagai HTML5.
  • <html>: Tag utama yang membungkus seluruh konten halaman.
  • <head>: Berisi informasi tentang halaman (meta data, judul, link CSS).
  • <body>: Berisi semua konten yang terlihat oleh pengguna (teks, gambar, link).

Contoh Kode HTML Sederhana:

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Halaman Pertama</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah halaman HTML pertama saya.</p>
</body>
</html>

Pertemuan 1: Pengenalan HTML & Struktur Dasar Website

Topik yang Dibahas

  • Apa itu HTML?
  • Sejarah dan Fungsi HTML
  • Struktur Dasar Dokumen HTML
  • Tag Dasar HTML (heading, paragraf, gambar, link)
  • Cara Menyimpan dan Membuka File HTML

Contoh Struktur HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Halaman Pertama Saya</title>
</head>
<body>
    <h1>Selamat Datang di Website Saya!</h1>
    <p>Ini adalah contoh paragraf pertama.</p>
    <a href="https://www.google.com">Kunjungi Google</a>
</body>
</html>

Tugas Praktikum

Buatlah file HTML dengan nama halaman-saya.html yang berisi:

  1. Judul halaman: "Profil Saya"
  2. Heading 1 dengan nama kamu
  3. 2 paragraf tentang diri kamu
  4. Satu gambar profil (gunakan gambar dari folder assets/images)
  5. Link ke media sosial kamu (opsional)
