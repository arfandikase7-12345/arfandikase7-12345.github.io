Pengenalan JavaScript untuk Interaktivitas

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website menjadi interaktif (misal: tombol yang merespon klik, form yang bisa dicek, animasi).

Cara Menambahkan JavaScript ke HTML:

  • Inline JS: Ditambahkan ke tag HTML dengan atribut onclick atau onload.
  • Internal JS: Ditulis di dalam tag <script> di bagian <body>.
  • External JS: Ditulis di file terpisah (misal script.js) dan dihubungkan dengan tag <script>.

Tips Praktis Belajar JavaScript

  • Gunakan Console.log() untuk mengecek nilai variabel dan debug kode
  • Pelajari DOM terlebih dahulu agar mudah membuat interaksi dengan elemen HTML
  • Coba buat proyek kecil seperti kalkulator sederhana atau form validasi
  • Gunakan Local Storage untuk menyimpan preferensi pengguna atau data sementara
  • Selalu tulis kode dengan indentasi yang rapi agar mudah dibaca

Tugas Praktikum

Buatlah aplikasi interaktif sederhana dengan JavaScript:

  1. Buat form input untuk memasukkan nama dan umur pengguna
  2. Tambahkan tombol "Tampilkan Data" yang akan menampilkan inputan di halaman
  3. Tambahkan tombol "Ubah Tema" yang bisa mengganti warna latar belakang halaman
  4. Simpan data nama pengguna ke Local Storage agar tetap ada meskipun halaman di-refresh
  5. Simpan kode JS di file script-interaktif.js di folder js/
