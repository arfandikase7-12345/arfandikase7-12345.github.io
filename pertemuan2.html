Dasar CSS untuk Mengatur Tampilan Website

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan format halaman HTML (warna, ukuran, posisi elemen).

Cara Menambahkan CSS ke HTML:

  • Inline CSS: Ditambahkan langsung ke tag HTML dengan atribut style.
  • Internal CSS: Ditulis di dalam tag <style> di bagian <head>.
  • External CSS: Ditulis di file terpisah (misal style.css) dan dihubungkan dengan tag <link>.

Contoh Kode CSS Sederhana:

/* External CSS di style.css */
body {
    background-color: #f8f9fa;
    font-family: Arial, sans-serif;
}

h1 {
    color: #0066cc;
    text-align: center;
}

p {
    font-size: 16px;
    line-height: 1.5;
}

Pertemuan 2: Dasar CSS untuk Mengatur Tampilan Website

Topik yang Dibahas

  • Apa itu CSS dan fungsinya?
  • 3 Cara Menyisipkan CSS (Inline, Internal, External)
  • Selektor CSS Dasar (Elemen, Kelas, ID)
  • Properti Dasar: Warna, Teks, Margin, Padding
  • Cara Menggunakan External CSS File

Contoh Penggunaan CSS

File style.css:

/* Style untuk body */
body {
    font-family: Arial, sans-serif;
    background-color: #f8f9fa;
    margin: 0;
    padding: 0;
}

/* Style untuk kelas */
.judul {
    color: #27ae60;
    text-align: center;
    padding: 20px;
}

/* Style untuk ID */
#paragraf-utama {
    font-size: 18px;
    line-height: 1.6;
    margin: 0 20px;
}

File index.html (menggunakan external CSS):

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Dasar CSS</title>
    <link rel="stylesheet" href="../css/style.css">
</head>
<body>
    <h1 class="judul">Belajar CSS Dasar</h1>
    <p id="paragraf-utama">Ini adalah contoh paragraf dengan style CSS.</p>
</body>
</html>

Tips Penting

  • Gunakan external CSS untuk proyek besar agar mudah dikelola
  • Perhatikan prioritas selektor: ID > Kelas > Elemen
  • Gunakan warna yang sesuai dengan tema website agar tidak menyilaukan mata
  • Selalu tambahkan box-sizing: border-box; agar ukuran elemen lebih mudah diatur

Tugas Praktikum

Modifikasi file halaman-saya.html dari tugas sebelumnya dengan menambahkan external CSS:

  1. Buat file style-halaman.css di folder css/
  2. Atur warna latar belakang, warna teks, dan jenis font untuk seluruh halaman
  3. Beri style pada heading agar lebih menarik
  4. Atur ukuran dan posisi gambar profil
  5. Tambahkan margin dan padding agar konten tidak menempel di tepi layar
